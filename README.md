# video-preview
For the video preview assignment, I used an existing code to utilize the (onmouseover) and (onmouseout) events. I first downloaded the provided code and picked various videos from pexels.com to use. Then I inserted videos into video tags in HTML. 

To make the website more interactive, I included hover events that enable the videos to play and pause when the mouse hovers over them. Then I used the method .play () with onmouseover and the method .pause () on mouse out.
